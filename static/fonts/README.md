# Fonts

Place these font files here so the theme CSS can load them:

- `Bangers-Regular.ttf`
- `Megrim-Regular.ttf`
- `MajorMonoDisplay-Regular.ttf`
- `ComicRelief-Regular.ttf`

Hugo copies everything in the theme’s `static/` to the site root, so they will be served at `/fonts/...`.

You can also place them in your **site’s** `static/fonts/` to override the theme.
